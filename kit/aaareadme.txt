                Read Before Installing Message Exchange $VER$

          Please read the following important information before
          installing or using MX $VER$.

          Installation Information

          MX $VER$ requires the following:

          o  For VAX systems: OpenVMS VAX V5.5-2 or higher
          o  For Alpha systems: OpenVMS Alpha V6.0 or higher
          o  For IA64 systems:  OpenVMS Industry Standard 64 V82. or higher

          o  For UUCP support: DECUS UUCP 2.1 or later
          o  For Spam Assassin support: Spam Assassin 3.3.1 or higher.

          Contents of this Kit

          This kit contains all of the materials necessary to
          install MX. 

          You will need approximately [tbd] megabytes of disk space
          to hold the installation kit, plus [tbd] megabytes to
          hold the source kit.

          The MX$V$.% files make up the installation kit,
          which you should install with VMSINSTAL. Manuals are
          included in the installation kit in PostScript,
          plain ASCII, HTML, and Bookreader versions, and are
          placed in the MX_ROOT:[DOC] directory during the
          installation. You can use BACKUP to retrieve them from
          the installation kit prior to installation:

              $ BACKUP MX$V$.L/SAVE *.*

          It is strongly recommended that you read all of the
          manuals carefully before to install MX.

          Release Notes Information

          The release notes for MX $VER$ contain important
          installation-related information as well as a summary
          of changes since the last release. Also included is
          information on known problems, documentation errors,
          and restrictions. The release notes can be retrieved
          by using VMSINSTAL OPTIONS N.

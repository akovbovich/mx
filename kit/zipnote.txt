Message Exchange (MX) is an electronic mail system for OpenVMS systems.
It supports the following operating systems and protocols:

    OpenVMS VAX V5.5-2
    OpenVMS Alpha V6.0
    OpenVMS Industry Standard 64 V8.2
    Internet SMTP    (requires a TCP/IP package)
    SMTP over DECnet (requires DECnet)

MX uses VMS Mail for local message storage.  MX also provide some
junk mail filtering settings and tools, and includes a  mailing list
processor and mail-based file server.
